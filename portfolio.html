  • Lulu's Recipe
    This website allows you to create and find recipes that you would want to learn or prepare. It also has the ability to save ingedients to a cart and convert it to PDF. This project was made using HTML/Bootstrap/Javascript. Jquery and ajax was used to pull API request.
  • Real-time Workday Scheduler
    This website allows you to log your activity per hour and marks gray if time has passed on that particular hour; red for current hour; and green for future hours. I used moment() for time and also used localStorage to save data for each input.
  • Weather Station
    This website allows you to search any cities for current weather. It can also provide 5 day forecast as well. I used ajax and jquery to build to logic in Javascript. I added some bootstrap for CSS as well.
  • Password Generator
    This website allows you to create a password by answering prompts to build your own unique password. Used vanilla javascript to build the site.
